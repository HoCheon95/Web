책갈피.html 책갈피 기능

책갈피 기능 구현하기

첫째 | 둘째 | 셋째 | 넷째

책갈피 기능을 구현하려면 다음과 같은 순서로 설정하고 실행하면 된다.

첫째, 문서 내에서 이동할 위치에 a태그를 설정하는데 a 태그의 id 속성에 고유한 이름을 지정한다.










































































top으로 이동

둘째, 링크를 설정할 때 href 속성에 '#id속성값' 형태로 지정한다.
여기서 'id속성값'은 첫쨰에서 설정한 id 속성값들 중 한가지를 말한다.











































































top으로 이동

셋째, 이제 둘째에서 만들어진 링크를 클릭하면 해당 위치로 바르게 이동하는 것을 확인할 수 있다.
